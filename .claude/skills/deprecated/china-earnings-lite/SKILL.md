# China Earnings Lite — 零依赖 A股财报综合分析

## 功能概述

**零外部依赖**，仅用 Python 标准库（`urllib`、`json`、`ssl`），直接请求公开 API 实现三大功能模块：

### 模块一：财务数据获取
通过东方财富 API 获取利润表、资产负债表、现金流量表数据，自动提取并计算关键指标。

### 模块二：财报 PDF 下载
通过巨潮资讯网（cninfo.com.cn）下载年报、半年报、季报原文 PDF。

### 模块三：行业基本面分析
获取所属行业板块行情、同行公司对比、行业资金流向，定位公司在行业中的位置。

### 模块四：综合报告生成
融合三大模块数据，生成详细的 Markdown 分析报告，包含：
- 公司概况与估值
- 关键财务指标总览（含同比变化）
- 利润表深度分析（收入利润、盈利能力、费用结构、历史趋势）
- 资产负债表分析（资产规模、资产结构、有息负债）
- 现金流量表分析（三大现金流、现金流质量判断）
- 行业对比分析（行业PE、同行排名、资金流向）
- 风险提示与投资亮点
- 综合评分与多维度评价

## 数据源

所有数据源均为**公开免费 API**，无需任何 token：

| 数据源 | 用途 | API 端点 |
|--------|------|----------|
| 东方财富 | 财务报表数据 | `datacenter-web.eastmoney.com` |
| 东方财富 | 个股行情/行业数据 | `push2.eastmoney.com` |
| 东方财富（备用）| 财务报表数据 | `datacenter.eastmoney.com` |
| 巨潮资讯 | 财报PDF下载 | `www.cninfo.com.cn` |

## 使用方法

### 前置条件

**仅需 Python 3.6+**，无需安装任何第三方库。

### 运行脚本

```bash
# 完整分析（数据 + 行业 + PDF下载 + 综合报告）
python {skill_path}/scripts/earnings_lite.py --code 300014

# 指定年份
python {skill_path}/scripts/earnings_lite.py --code 300014 --year 2024

# 仅获取财务数据（不下载PDF、不做行业分析）
python {skill_path}/scripts/earnings_lite.py --code 300014 --mode data

# 仅下载财报PDF
python {skill_path}/scripts/earnings_lite.py --code 300014 --mode download
python {skill_path}/scripts/earnings_lite.py --code 300014 --mode download --download-type half

# 仅行业分析
python {skill_path}/scripts/earnings_lite.py --code 300014 --mode industry

# 获取更多期数的历史数据
python {skill_path}/scripts/earnings_lite.py --code 300014 --periods 8

# 指定输出目录
python {skill_path}/scripts/earnings_lite.py --code 300014 --output-dir ./reports
```

### 命令行参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--code` | 股票代码（6位数字）| 必填 |
| `--year` | 分析年份 | 最新 |
| `--periods` | 获取历史期数 | 4 |
| `--mode` | 运行模式: data/download/industry/report | report |
| `--output-dir` | 输出目录 | 当前目录 |
| `--download-type` | PDF类型: annual/half/q1/q3 | annual |

### 输出文件

运行 `--mode report` 后生成：

| 文件 | 说明 |
|------|------|
| `{code}_income.json` | 利润表原始数据 |
| `{code}_balance.json` | 资产负债表原始数据 |
| `{code}_cashflow.json` | 现金流量表原始数据 |
| `{code}_analysis.json` | 结构化分析结果（含衍生指标、风险评估）|
| `{code}_industry.json` | 行业对比数据 |
| `{code}_{公司名}_综合分析报告.md` | 详细 Markdown 分析报告 |
| `{code}_*.pdf` | 财报 PDF 原文（来自巨潮资讯）|

### 分析报告内容结构

```
一、公司概况（代码、名称、行业、市值、估值）
二、关键财务指标总览（核心指标 + 同比 + 效率指标）
三、利润表分析
   3.1 收入与利润
   3.2 盈利能力（毛利率、净利率）
   3.3 费用结构（研发/销售/管理/财务费用及营收占比）
   3.4 历史趋势（多期对比表）
四、资产负债表分析
   4.1 资产规模与结构
   4.2 资产结构分布（各项占比）
   4.3 有息负债
五、现金流量表分析（三大现金流 + 质量判断）
六、行业对比分析
   6.1 行业整体情况（PE、涨跌幅、市值）
   6.2 行业资金流向
   6.3 同行公司对比（前10大公司排名表）
七、风险提示与投资亮点（自动识别）
八、综合评估（评分 + 多维度评价矩阵）
```

### 自动计算的衍生指标

| 指标 | 计算方式 | 需要数据 |
|------|----------|----------|
| ROE | 净利润/股东权益 | 利润表+资产负债表 |
| ROA | 净利润/总资产 | 利润表+资产负债表 |
| 总资产周转率 | 营收/总资产 | 利润表+资产负债表 |
| 应收账款周转率 | 营收/应收账款 | 利润表+资产负债表 |
| 存货周转率 | 营业成本/存货 | 利润表+资产负债表 |
| 经营现金流/净利润 | 经营现金流/净利润 | 现金流+利润表 |

## 风险自动识别规则

| 触发条件 | 级别 | 提示 |
|----------|------|------|
| 净利润同比 < -30% | 警示 | 盈利能力严重恶化 |
| 营收同比 < -15% | 警示 | 业务收缩明显 |
| 资产负债率 > 75% | 警示 | 杠杆过高 |
| 商誉/总资产 > 15% | 警示 | 商誉减值风险 |
| 经营现金流 < 0 | 警示 | 盈利质量存疑 |
| 净利率 < 0 | 警示 | 亏损状态 |
| ROE > 20% | 亮点 | 资本回报优秀 |
| 毛利率 > 50% | 亮点 | 定价能力强 |
| 营收增长 > 20% | 亮点 | 业务高速扩张 |

## 错误处理

1. **多级回退**: 东方财富主API → 东方财富备用API
2. **独立容错**: 每个模块独立运行，单模块失败不影响其他
3. **网络超时**: 每个请求 15 秒超时
4. **退出码**: 0=成功, 1=部分失败, 2=参数错误

## 与 china-earnings-analyzer 的区别

| 维度 | china-earnings-lite（本 skill） | china-earnings-analyzer |
|------|-------------------------------|------------------------|
| **依赖** | 零依赖（纯标准库） | 需要 akshare/tushare/efinance + pandas |
| **Python 版本** | 3.6+ | 3.9+ |
| **数据源** | 东方财富/巨潮公开 API | AkShare/Tushare 封装 |
| **PDF 下载** | 支持（巨潮资讯） | 不支持 |
| **行业分析** | 支持（同行对比、资金流向）| 不支持 |
| **报告深度** | 8大章节综合报告 + 评分 | 简单指标摘要 |
| **衍生指标** | ROE/ROA/周转率等自动计算 | 依赖数据源提供 |
| **启动速度** | 秒级 | akshare 导入可能 60s+ |
