## 三、行业基本面与竞争格局

### 3.1 煤制烯烃行业格局

**行业核心逻辑**：煤制烯烃的盈利取决于**油煤价差**。当原油价格处于中高位（>60美元/桶）时，煤制烯烃相对油制烯烃具有显著成本优势。

| 维度 | 现状 |
|------|------|
| **煤制vs油制成本差** | 2204元/吨（截至2026年3月） |
| **煤制烯烃平均成本** | 6573元/吨 |
| **油制烯烃平均成本** | 8777元/吨 |
| **宝丰市占率** | ~34%（行业第一） |
| **产能规模** | 520万吨/年 |

### 3.2 行业供需展望

**供给端**：
- 2024-2025年聚烯烃产能投放较多，行业竞争加剧
- **2026年以后新增产能有限**，供给端压力将显著缓解 `[中可信度]`
- 煤化工行业超6000亿投资潮主要集中在2025-2026年落地，但多数项目建设周期2-3年

**需求端**：
- 聚烯烃下游需求（包装、汽车、建材等）与宏观经济高度相关
- 2026年聚乙烯价格较去年上涨约10%（约800元/吨），聚丙烯上涨约1000元/吨
- 预计2026年全年价格将适当回归理性，但中枢仍高于2024年

**煤炭成本端**：
- 2026年煤炭供需维持相对宽松状态，新能源发电持续替代火电
- 煤价下行有利于宝丰能源成本端进一步改善

### 3.3 竞争优势分析

| 优势维度 | 具体表现 |
|---------|---------|
| **规模壁垒** | 520万吨/年产能，全国第一，内蒙古基地全球单厂最大 |
| **成本优势** | 自有煤矿+西部低价煤+规模效应，吨成本行业最低水平 |
| **一体化布局** | 煤炭开采→煤化工→精细化工完整产业链 |
| **区位优势** | 宁夏/内蒙古煤炭资源丰富，运输成本低 |
| **绿氢布局** | 光伏制氢耦合煤化工，降低碳排放，抢占政策先机 |
| **税收优惠** | 西部大开发15%企业所得税率（vs标准25%） |

## 六、前瞻研判：三情景分析

### 基准情景（概率50%）

**核心假设**：原油价格维持60-80美元/桶区间，煤价温和下行，聚烯烃价格小幅回落但仍高于2024年水平。

- **2026E净利润**：115-125亿元（同比持平或小幅增长）
- **逻辑**：内蒙古项目已满产运行，产能增量有限；产品价格温和回落但成本端受益于煤价下行；资产负债率继续下降，财务费用减少
- **对应合理估值**：24-28元（15倍PE）
- **证伪条件**：2026Q1净利润低于25亿元（低于2025Q1的24.37亿）

### 乐观情景（概率25%）

**触发条件**：地缘政治推升油价至80美元/桶以上，油煤价差进一步扩大；或下游需求超预期复苏。

- **2026E净利润**：130-145亿元（同比+15%~+28%）
- **潜在收益空间**：股价可看至33-38元（17倍PE）
- **证伪条件**：Brent原油价格持续低于60美元/桶

### 悲观情景（概率25%）

**风险触发点**：全球经济衰退导致聚烯烃需求大幅萎缩；或煤价大幅反弹压缩利润空间。

- **2026E净利润**：85-100亿元（同比-12%~-25%）
- **下行幅度**：股价可能回落至18-22元（13倍PE）
- **早期预警信号**：聚乙烯/聚丙烯价格连续3个月下跌超过10%

## 八+、重大政策事件影响专题：《碳达峰碳中和综合评价考核办法》

> **政策发布日期**：2026年4月23日（中共中央办公厅、国务院办公厅联合印发）
> **审批层级**：2026年2月26日中共中央政治局常委会会议审议批准
> **生效范围**：自2026年度起对各省（自治区、直辖市）实施评价考核

### 8+.1 政策核心要点

本办法是"双碳"战略落地的**最高级别考核制度**，由政治局常委会审批、两办联合印发，政策权威性和执行力度极强：

| 维度 | 关键内容 |
|------|---------|
| **考核对象** | 各省（自治区、直辖市）党委和政府，实行**党政同责** |
| **控制指标** | 碳排放总量、碳排放强度、**煤炭消费总量**、**石油消费总量**、非化石能源消费占比 |
| **支撑指标** | 节能、工业、城乡建设、交通运输、碳排放权交易等领域指标 |
| **核心目标** | 2030年碳排放强度比2005年降低65%以上、非化石能源消费占比达25%、**煤炭和石油消费总量达峰** |
| **考核等次** | 优秀/合格/不合格三等，控制指标不达标即"不合格" |
| **结果运用** | 与干部选拔任用、监督管理挂钩；不合格须30个工作日书面整改；可追责 |
| **关键新要求** | 新（改、扩）建"两高"工业项目须实施**碳排放等量或减量置换** |

### 8+.2 对宝丰能源的影响分析

#### 一、短期压力：碳排放合规成本将实质性上升 `[高可信度]`

宝丰能源的核心业务——**煤制烯烃**——属于典型的高碳排放行业（"两高"项目），是此次政策精准打击的对象：

| 压力维度 | 具体影响 |
|---------|---------|
| **煤炭消费总量管控** | 政策明确要求"煤炭消费总量达峰"，并将其列为**控制指标**（不达标即考核"不合格"）。各省将被迫严控煤炭消费增量，宝丰所在的宁夏和内蒙古作为煤化工重镇，将面临最直接的总量约束 |
| **碳排放等量/减量置换** | 新（改、扩）建"两高"项目须实施碳排放等量或减量置换，意味着宝丰未来的**产能扩张将受到实质性限制**，获取新增煤化工产能审批的难度和成本将大幅提高 |
| **碳市场成本** | 碳排放权交易被列为支撑指标，碳市场覆盖范围和价格机制将加速完善，宝丰作为碳排放大户，碳配额购买成本将显著增加 |
| **地方政府压力传导** | 考核实行"党政同责"并与干部任免挂钩，地方政府对辖区内高碳排放企业的监管和约束力度将空前加强 |

**量化影响估计**：以煤制烯烃吨产品碳排放约11吨CO₂计算，520万吨产能对应年碳排放约5700万吨。若碳价从当前约80元/吨上升至100-150元/吨（政策驱动下的合理预期），碳成本增量可达**11-40亿元/年**，对应压缩净利润约10%-35%。`[中可信度]`

#### 二、中期利好：存量产能的"护城河"价值大幅提升 `[高可信度]`

这是一个**典型的"利空增量、利好存量"**的政策：

| 利好维度 | 具体逻辑 |
|---------|---------|
| **产能壁垒加固** | 碳排放等量/减量置换要求将使新增煤化工产能审批极为困难，宝丰现有的520万吨/年产能成为**稀缺性资产**，竞争格局将显著改善 |
| **供给端出清加速** | 高碳排放的中小煤化工企业将面临更大的合规压力，行业集中度有望进一步向龙头倾斜，宝丰的市占率（目前34%）有望被动提升 |
| **产品价格支撑** | 供给端收缩将支撑聚烯烃价格中枢，部分对冲碳成本上升的压力 |
| **先发优势兑现** | 宝丰作为行业内最早布局绿氢的企业，其光伏制氢耦合煤化工的战略前瞻性将获得**巨大的政策红利** |

#### 三、长期战略：绿氢布局从"可选项"变为"生存线" `[高可信度]`

政策明确要求"**力争年度新增清洁能源电量逐步覆盖全社会新增用电量**"、"非化石能源消费占比达25%"，这直接提升了宝丰绿氢战略的价值：

| 战略维度 | 分析 |
|---------|------|
| **绿氢替代灰氢** | 宝丰已建成全球最大的电解水制绿氢项目（2万标方/时），用绿氢替代煤制氢环节可**显著降低碳排放强度**，在碳排放考核中获得优势 |
| **碳排放强度下降** | 绿氢耦合煤化工可将吨产品碳排放从约11吨降至7-8吨，在碳交易体系中释放大量**碳配额盈余**，从碳成本负担者转为碳资产持有者 |
| **估值逻辑重塑** | 随着碳成本纳入行业定价体系，宝丰的"煤化工+绿氢"模式可能获得**绿色溢价**，PE估值中枢从煤化工的13-17倍向绿色化工的18-22倍迁移 |
| **政策补贴预期** | 绿氢在碳排放考核体系中具有正向贡献，未来可能获得碳减排奖励、绿色信贷优惠等政策支持 |

## 九、结构化结论

### 核心判断

**宝丰能源是A股煤化工赛道最优质的标的，基本面极为扎实，但当前股价（~28元）已较充分反映了2025年的优异业绩，短期估值偏合理偏贵。最佳买入策略是等待股价回调至25元以下再分批建仓。** `[中可信度]`

### 判断依据

1. 2025年净利润113.5亿，同比+79%，经营现金流168.5亿，盈利质量极高 `[高可信度]`
2. 煤制烯烃产能520万吨/年，行业第一，成本优势显著 `[高可信度]`
3. 当前PE约20倍，对于周期股而言处于合理偏高水平 `[高可信度]`
4. 股价从历史高点36.49元回调22%，但尚未到达理想买入区间 `[高可信度]`
5. 2026年聚烯烃新增产能有限，供给端压力缓解有利于价格维持 `[中可信度]`
6. 2026年4月《碳达峰碳中和综合评价考核办法》印发，短期增加碳合规成本压力，但中长期利好龙头存量产能的稀缺性和绿氢布局的战略价值 `[高可信度]`

### 主要假设

- 2026年原油价格维持在60-80美元/桶区间
- 煤炭价格不会大幅反弹
- 内蒙古基地产能利用率维持在85%以上
- 不发生重大安全生产事故
- 碳排放权交易价格在2026年不超过120元/吨（若大幅超预期上涨则需重估盈利）
