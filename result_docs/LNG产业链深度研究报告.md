## 一、产业链全景概览

### 1.1 LNG产业链结构

LNG（Liquefied Natural Gas，液化天然气）产业链是一条从天然气勘探开采到终端消费的完整价值链，可分为**上游、中游、下游**三大环节：

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        LNG 产业链全景图                                  │
├──────────────┬──────────────────────┬────────────────────────────────────┤
│    上  游     │       中  游          │            下  游                  │
├──────────────┼──────────────────────┼────────────────────────────────────┤
│ 天然气勘探    │ LNG液化加工           │ 城市燃气（居民/商业）               │
│ 天然气开采    │ LNG接收站（再气化）    │ 工业燃料                          │
│ 煤层气开发    │ LNG储罐/储运          │ 天然气发电                        │
│ 页岩气开发    │ LNG运输船             │ 交通燃料（LNG重卡/船舶加注）        │
│ 煤制天然气    │ 天然气管网             │ 化工原料（甲醇/合成氨等）           │
│ 海外资源获取  │ LNG槽车运输           │ LNG加气站                         │
│              │ LNG装备制造           │ 分布式能源                        │
└──────────────┴──────────────────────┴────────────────────────────────────┘
```

### 1.3 行业驱动因素

- **"双碳"政策驱动**：天然气作为最清洁的化石能源，是煤炭替代的核心过渡能源，"十四五"规划明确提高天然气消费占比
- **能源安全战略**：国家持续推进"增储上产"七年行动计划，加大国内天然气勘探开发力度
- **管网改革红利**：国家管网集团成立后，"X+1+X"油气市场体系初步形成，市场化改革加速
- **LNG重卡替代**：油气价差驱动LNG重卡渗透率快速提升，2025年保有量预计增至95万辆
- **全球LNG扩产周期**：2026-2030年全球预计新增约2.02亿吨LNG产能，供应格局重构

## 二、上游环节：资源勘探与天然气生产

### 2.1 业务形态

上游环节涵盖天然气的**勘探、开采、液化**等核心环节，是整个产业链的资源基础：

- **常规天然气勘探开采**：陆上/海上油气田的勘探与开发
- **非常规天然气**：煤层气（CBM）、页岩气、致密气的开发利用
- **煤制天然气**：以煤炭为原料，通过气化工艺生产合成天然气
- **海外资源获取**：通过长协合同、权益投资等方式获取海外LNG资源
- **天然气液化**：将天然气冷却至-162°C液化，体积缩小至1/600

## 三、中游环节：储运、液化与装备制造

### 3.1 业务形态

中游环节是连接上游资源与下游消费的关键纽带，包括：

- **LNG接收站**：进口LNG的接卸、储存、再气化，是LNG进口的核心基础设施
- **天然气管网**：骨干管道（西气东输、中俄东线等）和区域管网
- **LNG运输**：远洋LNG运输船、内河LNG船、LNG槽车
- **LNG储罐**：大型低温储罐的设计、制造与安装
- **LNG装备制造**：液化设备、气化设备、低温阀门、压缩机等

### 3.2 基础设施现状

| 基础设施 | 现状 | 发展趋势 |
|---------|------|---------|
| **LNG接收站** | 全国已投产28座，总接收能力超1.4亿吨/年 | 在建和规划约5,000万吨/年 |
| **天然气管道** | 总里程超12.8万千米 | 西气东输四线、川气东送二线推进中 |
| **中俄东线** | 2024年全线贯通，年输气能力380亿方 | 2025年向长三角增供近50亿方 |
| **LNG运输船** | 沪东中华2025年交付11艘，累计承接超100艘 | 2026年订单有望爆发 |

## 四、下游环节：终端应用与消费

### 4.1 业务形态

下游环节是天然气的终端消费市场，主要包括：

| 应用领域 | 占比 | 增长驱动 |
|---------|------|---------|
| **工业燃料** | ~43% | 煤改气政策推动，工业用气需求稳定 |
| **城市燃气** | ~30% | 城镇化推进，居民用气价格机制理顺 |
| **天然气发电** | ~15% | 电力市场化改革，调峰需求增长 |
| **交通燃料** | ~8% | LNG重卡渗透率快速提升，船舶加注兴起 |
| **化工原料** | ~4% | 甲醇、合成氨等化工品生产 |

## 六、产业链整体研判

### 6.1 当前市场环境分析

#### 📌 地缘政治：中东冲突深刻重塑LNG供应格局

2026年2-3月，美以与伊朗冲突升级，霍尔木兹海峡事实上被封锁，对全球LNG市场产生了**颠覆性影响**：

- **卡塔尔LNG出口受阻**：卡塔尔生产全球约20%的LNG，霍尔木兹海峡封锁后出口中断，拉斯拉凡工业城遇袭导致约17%产能停止
- **气价飙升**：东北亚JKM从2月底的约13美元/百万英热飙升至30美元以上，欧洲TTF上涨52.6%
- **供应格局重构**：全球LNG供应从"买方市场"预期急转为"供应紧张"，美国LNG出口重要性凸显

#### 📌 供给侧：全球LNG进入"超级扩张周期"

- 2026年全球预计新增约3,500万吨/年LNG产能，主要来自美国和卡塔尔
- 2026-2030年累计新增约2.02亿吨产能，较2025年增长约40%
- 但中东冲突使卡塔尔扩产进度存在重大不确定性

#### 📌 需求侧：中国天然气消费稳健增长

- 预计2025年中国天然气表观消费量增速6-7%，较2024年的9-10%有所放缓
- 工业用气约1,767亿方，与2024年持平
- LNG重卡渗透率持续提升，交通用气成为增量亮点
- AI驱动的数据中心电力需求增长，天然气发电前景广阔

**LNG产业链正处于"短期波动、中期扩张、长期向好"的关键阶段**：

2. **中期（2027-2028年）**：全球LNG进入"超级扩张周期"，新增产能集中释放，供需格局可能从紧平衡转向宽松。**中游基础设施和装备制造环节确定性更强**，不受气价波动影响。

3. **长期（2029-2030年）**：天然气作为"双碳"目标下的过渡能源，消费占比将持续提升。**城燃龙头和全产业链一体化公司将享受行业成长红利**，估值有望系统性重估。
