## 第一层：财务数据深度拆解

### 1.1 收入规模与增长质量

| 指标 | 赣锋锂业 | 华友钴业 | 对比分析 |
|------|---------|---------|---------|
| **2025全年营收** | 230.82亿 | 810.19亿 | 华友是赣锋的3.5倍 |
| **2025营收同比** | +22.08% | +32.94% | 华友增速更快 |
| **2026Q1营收** | 未披露（仅预告利润） | 258.04亿（+44.62%） | 华友单季营收已超赣锋全年的1/4 |
| **营收结构** | 锂盐为主+锂电池 | 镍钴锂+前驱体+正极材料 | 华友更多元化 |

**关键发现**：华友钴业的营收体量是赣锋锂业的3.5倍，这源于两者商业模式的根本差异——华友是"资源+冶炼+材料"的纵向一体化模式，营收包含大量中间品加工环节的流转；赣锋则更偏上游资源+锂盐加工。**营收规模差异不等于盈利能力差异**，需要看利润率。 [高可信度]

### 1.2 利润质量深度对比

| 指标 | 赣锋锂业 | 华友钴业 | 解读 |
|------|---------|---------|------|
| **2025全年净利** | 16.13亿 | 61.10亿（+47.07%） | 华友绝对利润是赣锋3.8倍 |
| **2025净利率** | 6.99% | 7.54% | 接近，华友略高 |
| **2026Q1净利** | 16-21亿（预告） | 24.97亿 | 华友更高 |
| **2026Q1扣非净利** | 12.5-17.5亿 | 20.70亿（+68.76%） | 华友扣非也更高 |
| **Q1非经常损益** | ~3.5亿（PLS股票公允价值+领式期权，约2.59亿） | 4.27亿（资产处置4.81亿+政府补助0.20亿-公允价值损失0.36亿） | 两者都有较大非经常损益 |
| **2025毛利率** | 15.72%（+2.77pp） | 17.47%（全年） | 华友更高 |
| **2026Q1毛利率** | 未披露 | 20.69%（+4.83pp） | 华友毛利率持续上行 |

**关键发现**：

1. **华友的利润绝对值确实远超赣锋**，2025全年是3.8倍，2026Q1仍然领先约30-50%。这是"华友业绩更好"这一判断的核心数据支撑。 [高可信度]

2. **但赣锋的利润弹性更大**：赣锋从2025Q1亏损3.56亿到2026Q1盈利16-21亿，是从亏损到暴利的拐点；华友则是从盈利12.52亿到24.97亿，是稳步增长。**周期股投资中，拐点的弹性往往比稳态增长更受关注**。 [中可信度]

3. **两者的非经常损益占比都不低**：赣锋Q1约2.59亿来自PLS股票，华友约4.27亿来自资产处置。扣非后华友的优势更加明显（20.70亿 vs 12.5-17.5亿）。 [高可信度]

### 1.3 资产负债结构对比

| 指标 | 赣锋锂业 | 华友钴业 | 评价 |
|------|---------|---------|------|
| **总资产** | 1,132.58亿 | 1,746.69亿 | 华友资产更重 |
| **资产负债率** | 54.23% | 63.57% | 赣锋更稳健 |
| **固定资产占比** | 18.74% | 30.07% | 华友重资产特征更明显 |
| **存货占比** | 11.48%（130亿） | 17.35%（303亿） | 华友存货压力更大 |
| **货币资金** | 84.22亿（7.44%） | 223.35亿（12.79%） | 华友现金更充裕 |
| **应收账款** | 67.72亿 | 115.56亿（Q1增23.70%） | 华友应收增速偏快 |

**关键发现**：

1. **华友的资产负债率63.57%处于中高水平**，且Q1短期借款达298.57亿（+19.19%），筹资活动现金流净流入79.11亿（+1089%），说明公司仍在大规模举债扩张。这是一把双刃剑——扩张期的高杠杆在景气上行期放大收益，但在下行期也会放大风险。 [高可信度]

2. **华友的存货303亿（+18.29%）值得警惕**：在金属价格回升期备货是合理的，但如果后续价格回落，存货跌价风险不容忽视。 [中可信度]

3. **赣锋的资产负债率54.23%相对稳健**，但也需注意其财务费用高达15.34亿（占营收6.65%），说明有息负债规模不小。 [高可信度]

### 1.4 现金流健康度对比

| 指标 | 赣锋锂业 | 华友钴业 | 评价 |
|------|---------|---------|------|
| **经营现金流** | 29.45亿（+786%） | 11.75亿（-70.72%） | 赣锋大幅改善，华友恶化 |
| **经营现金流/净利润** | 1.83倍 | 0.47倍 | **赣锋含金量远超华友** |
| **投资现金流** | -44.07亿 | -44.64亿 | 两者都在大规模投资 |
| **筹资现金流** | +37.20亿 | +79.11亿（+1089%） | 华友融资力度更大 |
| **自由现金流** | 约-14.62亿 | 约-32.89亿 | 两者都为负 |

**关键发现**：

1. **这是赣锋最大的亮点之一**：经营现金流/净利润达1.83倍，说明赣锋的利润是"真金白银"，盈利含金量高。而华友仅0.47倍，意味着近一半的利润尚未转化为现金流入。 [高可信度]

2. **华友经营现金流同比下降70.72%是一个值得关注的信号**：虽然营收增长44.62%，但经营现金流反而大幅下降，主要原因是应收账款和存货的大幅增加占用了大量资金。这在扩张期是常见的，但需要持续跟踪。 [高可信度]

3. **华友的筹资现金流暴增1089%**，取得借款209.14亿（+100.61%），说明公司正在通过大规模举债支撑产能扩张。 [高可信度]

### 1.5 杜邦分析：ROE来源拆解
- :eyes: Q1单季ROE 5.05%，全年ROE 10.05%处于合理水平
**关键发现**：华友2025全年ROE达10.05%，远超赣锋的3.11%，且呈逐年上升趋势（2023年7.49%→2024年8.54%→2025年10.05%）。华友更高的ROE来源于**更高的净利率**和**更高的杠杆（权益乘数2.74 vs 2.18）**。需要注意的是，高杠杆驱动的ROE在周期下行时会快速恶化，但华友ROE的持续改善趋势说明其一体化战略正在兑现。赣锋的ROE则刚从2024年的-4.36%转正至3.11%，处于恢复早期。 [高可信度]

## 第二层：业务模式与竞争力解读

### 2.1 赣锋锂业：全球锂资源龙头 + 固态电池先行者

**核心业务画像**：
- **上游资源**：全球布局锂矿（澳洲Pilbara权益、阿根廷Cauchari-Olaroz、马里Goulamina、墨西哥Sonora等），2025年生产18.24万吨LCE（+40%），目标产能27.7万吨LCE
- **中游加工**：锂盐产品线最齐全（碳酸锂、氢氧化锂、氯化锂、金属锂等40+种产品）
- **下游延伸**：锂电池板块（动力+储能），产销量明显提升
- **前沿布局**：固态电池硫化物路线，电解质产能5,000吨（全球最大），能量密度420Wh/kg

**竞争优势**：
1. 锂资源全球化布局最全，自有资源供应能力强
2. 产品线最齐全，能满足客户多元化需求
3. 固态电池布局领先，硫化物电解质+锂金属负极形成技术壁垒
4. 锂回收能力20万吨/年，形成闭环

**核心风险**：
1. 业绩高度依赖锂价，周期波动大
2. 海外矿山运营风险（政治、汇率、物流）
3. 固态电池商业化时间表不确定

### 2.2 华友钴业：镍钴锂一体化龙头 + 新能源材料平台

**核心业务画像**：
- **上游资源**：刚果（金）钴矿+印尼红土镍矿+津巴布韦锂矿，多金属布局
- **中游冶炼**：2025年镍产品出货29.25万金属吨（+58.72%），钴产品4.65万吨
- **下游材料**：三元前驱体出货10.8万吨（+5%），正极材料突破10万吨（+108%），全球市占率领先
- **客户绑定**：深度绑定宁德时代、LG等头部电池厂

**竞争优势**：
1. "矿-冶-材"一体化，产业链利润留存能力强
2. 多金属（镍+钴+锂+铜）分散单一品种价格风险
3. 前驱体全球市占率12%+，正极材料高速增长
4. 印尼镍项目成本优势显著

**核心风险**：
1. 刚果（金）钴出口管制政策不确定性
2. 印尼镍矿政策风险（基准价调整、出口限制）
3. 资产负债率偏高，扩张期资金压力大
4. 存货规模大（303亿），金属价格下跌时跌价风险高

### 2.3 商业模式本质差异

| 维度 | 赣锋锂业 | 华友钴业 |
|------|---------|---------|
| **定位** | 锂资源+锂盐专精 | 多金属一体化平台 |
| **收入弹性** | 高（锂价×量） | 中（多品种对冲） |
| **利润波动性** | 极高（2024亏20亿→2025赚16亿） | 中等（持续盈利，波动较小） |
| **增长驱动** | 锂价回升+产能释放+固态电池 | 量增（镍+正极材料）+结构优化 |
| **风险特征** | 单一品种集中风险 | 多品种分散+高杠杆风险 |

## 第三层：行业与竞争格局

### 3.1 锂行业：周期底部回升，但分歧加大

- **碳酸锂价格**：2026Q1均价15.44万元/吨，环比+74.86%，同比+103.7%。4月16日电池级碳酸锂报16.75万元/吨，已从2025年中的8万元/吨低点翻倍
- **供需格局**：2025年前三季度行业普遍亏损，大量高成本产能出清。2026年需求端受新能源车+储能双轮驱动，供给端收缩后供需缺口出现
- **行业周期定位**：从"萧条"转向"复苏"早期，锂价中枢上移但尚未回到2022年的疯狂水平
- **SQM高管预计2026年碳酸锂价格约15-18美元/公斤**（折合约11-13万元/吨），与当前16.75万元/吨存在一定分歧

### 3.2 镍钴行业：供给侧改革带来结构性机会

- **钴**：刚果（金）2026年钴出口配额较2024年缩减56%，全球钴供给收缩约56%，供需缺口超20%，钴价中枢上移
- **镍**：2025年全球镍供应381万吨 vs 需求360万吨，仍处于供过于求，但印尼政策调整（基准价修订）可能改变格局
- **前驱体/正极材料**：下游需求旺盛，高镍化趋势明确，华友在9系超高镍产品上技术领先

### 3.3 两者在各自赛道的竞争坐标

| 维度 | 赣锋锂业 | 华友钴业 |
|------|---------|---------|
| **行业地位** | 全球锂盐龙头之一（与天齐、ALB并列） | 全球钴镍一体化龙头，前驱体全球前三 |
| **市值排名** | 能源金属板块第1-2名 | 能源金属板块第3名 |
| **同行PE对比** | 显著高于天齐锂业等同行 | 低于多数新能源材料公司 |
| **护城河** | 资源储量+产品齐全+固态电池 | 一体化成本+客户绑定+多金属 |

## 第四层：交叉验证——"华友更好"的结论是否成立？

### 4.1 支持"华友业绩更好"的证据 [高可信度]

1. **绝对利润**：2025全年华友61.10亿 vs 赣锋16.13亿，华友是赣锋的3.8倍
2. **Q1利润**：华友24.97亿 vs 赣锋16-21亿（中值18.5亿），华友高出35-55%
3. **扣非利润**：华友20.70亿 vs 赣锋12.5-17.5亿（中值15亿），华友高出38%
4. **盈利稳定性**：华友连续8个季度盈利且持续增长；赣锋2024全年亏损20.74亿，2025上半年仍在亏损
5. **毛利率趋势**：华友从14.88%→20.69%持续上行；赣锋从5.97%→15.72%虽然改善但绝对水平仍低

### 4.2 支持"华友估值更低"的证据 [高可信度]

1. **PE(TTM)**：华友约20倍 vs 赣锋约44-50倍，赣锋是华友的2倍以上
2. **市值/净利润**：华友1,250亿/61.10亿=20.5倍；赣锋1,770亿/16.13亿=109.7倍（用2025全年）
3. **若用2026Q1年化**：华友1,250亿/(24.97×4)=12.5倍；赣锋1,770亿/(18.5×4)=23.9倍，华友仍然更低

### 4.3 但"赣锋不好"的结论需要修正 [中可信度]

**市场给赣锋更高估值的合理性分析**：

1. **利润弹性差异巨大**：
   - 赣锋Q1净利中值18.5亿，而2025Q1亏损3.56亿，**同比增幅超过600%**
   - 如果锂价继续上行至20万元/吨，赣锋全年净利可能达到80-100亿+，对应当前市值PE仅18-22倍
   - 华友的利润弹性相对有限，因为多金属对冲了单一品种的弹性

2. **固态电池期权价值**：
   - 赣锋是全球固态电池硫化物路线的领军者，电解质产能5,000吨全球最大
   - 2026年被视为固态电池商业化元年，赣锋的估值中包含了这一远期期权
   - 华友在固态电池领域没有显著布局

3. **周期股估值逻辑**：
   - 周期股在利润低谷时PE高（甚至亏损时无PE），在利润高峰时PE低
   - 赣锋当前PE高，恰恰说明市场认为它处于利润回升的早期阶段
   - 如果用2026全年预期利润（机构预测60-80亿）计算，PE降至22-30倍

4. **现金流质量**：
   - 赣锋经营现金流/净利润=1.83倍，盈利含金量高
   - 华友仅0.47倍，利润"含水量"较高

### 4.4 信号矛盾汇总

| 矛盾点 | 分析 | 更可信的一方 |
|--------|------|-------------|
| 华友利润更高但估值更低 | 市场定价的是未来利润增速差异，而非当前利润绝对值 | 两者都有道理 |
| 赣锋PE高但利润弹性大 | 周期股估值需要用远期利润而非当前利润 | 赣锋的高PE有合理性 |
| 华友营收增长但现金流恶化 | 扩张期常见，但需警惕应收+存货风险 | 现金流数据更可信 |
| 赣锋扭亏但ROE仍低 | ROE是滞后指标，需看趋势而非绝对值 | 趋势更重要 |

### 5.2 三情景分析

#### 赣锋锂业


## 第六层：结构化结论

### 结论一：华友钴业确实在当前时点呈现"业绩更好+估值更低"

**【核心判断】** 华友钴业2026Q1净利24.97亿（+99.45%），扣非20.70亿（+68.76%），无论绝对值还是增速都优于赣锋锂业的预告数据。同时华友PE(TTM)约20倍，仅为赣锋的一半。这一"业绩更好+估值更低"的判断在当前数据下是成立的。

**【信号可信度】** 高可信度——基于已发布的财务数据

**【判断依据】**
1. 华友Q1净利24.97亿 > 赣锋预告中值18.5亿
2. 华友PE(TTM)~20倍 < 赣锋PE(TTM)~44-50倍
3. 华友2025全年净利61.10亿 = 赣锋的3.8倍
4. 华友毛利率20.69%持续上行，盈利能力改善趋势明确

**【主要假设】** 当前金属价格维持稳定，华友的一体化业务持续放量

**【证伪条件】** 若赣锋2026Q1实际净利超过21亿上限（即超预告上限），或锂价突破20万元/吨使赣锋Q2利润大幅超越华友

**【不确定性说明】** 置信度80%。赣锋Q1仅为预告（16-21亿区间），实际数据可能偏上限

### 结论二：赣锋锂业并非"不好"，而是处于不同的周期位置

**【核心判断】** 赣锋的高估值定价的不是当前利润，而是**锂价上行周期的利润弹性**和**固态电池的远期期权价值**。从2024年亏损20.74亿到2026Q1单季盈利16-21亿，赣锋正处于业绩拐点的爆发期，其利润增速（+549%~690%）远超华友（+99.45%）。

**【信号可信度】** 中可信度——基于趋势判断和周期逻辑

**【判断依据】**
1. 赣锋Q1同比增幅+549%~690%，是典型的周期拐点特征
2. 碳酸锂价格从8万→16.75万元/吨，仍在上行通道
3. 赣锋经营现金流/净利润=1.83倍，盈利含金量高于华友
4. 固态电池硫化物路线全球领先，电解质产能5,000吨
5. 机构预测赣锋2026全年净利60-80亿，对应PE 22-30倍

**【主要假设】** 锂价维持在15万元/吨以上，固态电池商业化持续推进

**【证伪条件】** 碳酸锂价格跌破12万元/吨持续2个月以上；或固态电池量产延迟至2028年以后

**【关键观察指标】** 碳酸锂期货主力合约价格（月度跟踪）、赣锋Q2实际净利润（7月披露）

**【不确定性说明】** 置信度65%。锂价走势存在较大不确定性，固态电池商业化进度难以精确预判

### 结论三：两者的投资逻辑本质不同——确定性 vs 弹性

**【核心判断】** 选择华友还是赣锋，本质上是在**确定性溢价**和**弹性溢价**之间做选择：
- **华友 = 确定性**：多金属一体化分散风险，利润稳定增长，估值合理，适合追求稳健回报的投资者
- **赣锋 = 弹性**：纯锂标的+固态电池期权，利润弹性极大，但波动也大，适合看好锂价上行+固态电池的投资者

**【信号可信度】** 中可信度——基于商业模式分析

**【判断依据】**
1. 赣锋利润对锂价的敏感度远高于华友（锂价每涨1万/吨，赣锋利润增10-15亿）
2. 华友的多金属布局天然对冲了单一品种风险
3. 华友的正极材料业务（+108%）提供了量增逻辑，不完全依赖价格
4. 赣锋的固态电池是独有的期权价值，华友没有

**【证伪条件】** 若华友也开始布局固态电池并取得突破，或赣锋的多元化（锂电池板块）贡献利润超过30%
