### 2.1 第一层：晶圆代工（最上游瓶颈环节）

#### 中芯国际（688981.SH）—— 独家代工，核心瓶颈 ⭐⭐⭐⭐

**参与深度：极深（独家）**

| 维度 | 详情 |
|------|------|
| **角色** | 昇腾950系列独家晶圆代工 |
| **制程** | N+2工艺（类7nm），良率已提升至92% |
| **产能** | 2026全年目标80-100万片，当前产能爬坡期 |
| **绑定程度** | 华为14nm/7nm/N+3全系列独家代工 |
| **核心壁垒** | 国内唯一具备7nm量产能力的代工厂；华虹N+2良率仅35-37%，无法形成有效补充 |

**研判**：中芯国际是昇腾950的**绝对瓶颈环节**。92%的良率在国产先进制程中已属顶尖，但总有效产能天花板框定了2026年国产高端AI芯片的供应上限。N+3（准5nm）的950DT芯片预计2026年流片，将进一步拉动先进制程需求。**确定性最高**。

**去伪存真**：市场一度传言华虹也参与代工，但实际数据显示华虹N+2良率过低（35-37%），根本无法量产。中芯国际的独家地位短期内无法动摇。

### 2.3 第三层：HBM存储（战略自主环节）

#### 华为自研HBM（HiBL 1.0）—— 战略级突破 ⭐⭐⭐⭐

**核心事实**：昇腾950PR采用华为自研HBM——HiBL（High Bandwidth Link）1.0，这是中国AI芯片产业**摆脱存储卡脖子**的标志性事件。

**产业链影响**：
- 自研HBM意味着华为需要自己的HBM封装产线，或与封测厂深度合作
- 存储颗粒的来源是关键问题——大概率与长鑫存储合作

#### 长鑫存储（未上市，母公司睿力集成）—— HBM颗粒供应 ⭐⭐⭐

| 维度 | 详情 |
|------|------|
| **角色** | 国内唯一具备先进DRAM量产能力的企业 |
| **HBM进展** | 据SemiAnalysis报告，2026年有望量产HBM3E |
| **与华为关系** | 高度协同，为华为自研HBM提供DRAM颗粒 |

**研判**：长鑫存储未上市是最大遗憾，但其母公司睿力集成及关联公司值得关注。长鑫在DRAM技术上已实现国产突破，HBM3E的量产将彻底打通国产AI芯片的存储链路。

#### 华海诚科（688535.SH）—— HBM封装GMC塑封料唯一供应 ⭐⭐⭐⭐

**参与深度：极深（唯一供应商）**

| 维度 | 详情 |
|------|------|
| **角色** | HBM先进封装用GMC（Granular Molding Compound）塑封料**国内唯一量产供应商** |
| **背景** | 华为哈勃已入股，深度绑定 |
| **壁垒** | GMC塑封料技术壁垒极高，国内仅此一家量产 |
| **扩产** | 并购衡所华威，年产销量突破2.5万吨，跃居全球第二 |

**研判**：华海诚科是昇腾950供应链中**隐形冠军级**标的。HBM封装对塑封料的要求极高（低翘曲、低应力、高流动性），GMC是关键材料。华为哈勃入股的信号非常明确——这是被资本层面绑定的供应商。**确定性极高，且市场认知度相对不足**。

## 四、关键研判与风险提示

### 4.1 核心研判结论

1. **最确定的环节是"卡脖子"环节**：中芯国际（代工）、长电科技（封装）、华海诚科（塑封料）——这三者是**无法替代、无法绕过**的核心供应商，确定性最高。

2. **被市场低估的隐形冠军**：
   - **华海诚科**：GMC塑封料唯一量产+华为哈勃入股，市场关注度远低于其供应链重要性
   - **华丰科技**：60-70%的连接器市占率，但市值相对偏低
   - **光迅科技**：液冷光模块独家（90%市占），这一细分市场被华工科技的光环所掩盖

3. **量最大的环节**：光模块（全光互联架构下用量成倍增长）和液冷（600W功耗刚性需求），是收入弹性最大的环节。

4. **华为自研HBM是最大变量**：HiBL 1.0的量产进度直接影响昇腾950的总出货量。长鑫存储的HBM3E量产进度是产业链的"X因素"。

### 4.2 需要去伪存真的传闻

| 传闻 | 真实性判断 | 理由 |
|------|-----------|------|
| "华虹也参与代工" | ❌ 不实 | 华虹N+2良率仅35-37%，无法量产 |
| "某厂商获得昇腾950独家大单" | ⚠️ 需审慎 | 除中芯/长电外，"独家"说法需验证 |
| "昇腾950对标B200" | ⚠️ 有误导 | 单卡对标H20更准确，950PR的FP8算力1P vs B200的9P仍有差距 |
| "订单排到2028" | ✅ 部分属实 | 核心互联网客户订单确实排至2027年+ |

### 4.3 风险提示

1. **地缘政治风险**：美国进一步制裁可能影响上游设备/材料供应
2. **中芯国际产能瓶颈**：良率92%已接近天花板，产能扩张需要时间
3. **HBM自研进度风险**：华为自研HBM尚处早期，量产稳定性待验证
4. **估值风险**：部分标的已被市场充分定价，需关注业绩兑现情况
5. **技术迭代风险**：950DT（Q4）和960（2027Q4）的推出可能改变供应链格局

## 五、总结

昇腾950系列是2026年中国AI算力产业的**绝对主线**。从全供应链视角看：

- **最深度参与**（独家/唯一/第一）：中芯国际、长电科技、华海诚科、华工科技、华丰科技、光迅科技
- **深度参与**（核心供应商）：深南电路、兴森科技、拓维信息、神州数码、通富微电
- **重要参与**（显著受益）：中际旭创、新易盛、英维克、申菱环境、东威科技、高新发展

投资逻辑的核心是：**沿着"不可替代性"和"订单可见性"两个维度筛选**。不可替代性越强（独家/唯一）、订单可见性越高（已锁定订单/公司公告确认），确定性越高。
