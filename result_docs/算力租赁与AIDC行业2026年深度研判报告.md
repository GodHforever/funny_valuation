## 一、行业全景：AI算力需求驱动的黄金赛道

### 1.1 行业驱动逻辑

2026年，全球AI产业进入**"Token经济"时代**，大模型Token消耗呈指数级增长，算力需求从"建设驱动"转向"消费驱动"。两大核心变量正在重塑行业格局：

- **需求端**：DeepSeek等国产大模型加速推进，Agent应用和多模态生态爆发，头部云厂商资本开支持续加码。2026年Q1，国内大模型Token消耗量同比增长超300%
- **供给端**：高端算力（H100/B200）持续紧缺，算力租赁价格普涨**20%-30%**，高端GPU涨幅达**30%-40%**。行业进入**量价齐升**周期

### 1.2 市场规模

| 指标 | 数据 |
|------|------|
| **2026年算力租赁潜在市场规模** | 2,600亿元 |
| **年均增速** | >20% |
| **"十五五"末AI产业规模目标** | >10万亿元 |
| **2026年全国算力基建投资** | 数千亿级 |

### 1.3 行业竞争格局演变

2026年行业呈现**"三化"趋势**：

- **头部化**：市场份额加速向头部供应商集中，仅头部云厂商能获得充足高端算力
- **技术化**：同质化竞争转向"技术+资源"双核心，国产适配、液冷技术成为关键壁垒
- **涨价化**：云涨价周期刚刚开始，随着Token使用量提升，高端算力将愈发紧缺

## 四、行业对比与投资框架

### 4.1 算力租赁 vs AIDC：商业模式差异

| 维度 | 算力租赁 | AIDC |
|------|---------|------|
| **商业模式** | 轻资产→重资产转型中 | 重资产运营 |
| **收入弹性** | 极高（量价齐升） | 中高（规模驱动） |
| **利润率** | 波动大（15-45%毛利率） | 相对稳定（30-50%毛利率） |
| **资本开支** | 快速增长期 | 持续高投入 |
| **客户集中度** | 较高 | 极高 |
| **技术壁垒** | 中（GPU采购+运维） | 高（能耗+选址+运维） |
| **当前景气度** | 🔴 极高 | 🟢 高 |

### 4.2 核心投资逻辑总结

```
算力租赁赛道：
  短期催化 → 算力租赁价格持续上涨（20-40%），Q1业绩超预期
  中期逻辑 → Token经济时代，算力消费从建设驱动转向需求驱动
  长期风险 → 国产替代进度、GPU供给放量后的价格压力

AIDC赛道：
  短期催化 → 头部公司AIDC业务收入占比快速提升，业绩验证
  中期逻辑 → 从概念叙事转向业绩兑现，龙头公司护城河加深
  长期风险 → 重资产模式下的资本回报率、能耗政策变化
```

## 六、结论与建议

### 6.1 行业判断

**算力租赁和AIDC行业正处于景气度上行周期的加速阶段**。Token经济时代的到来使得算力从"基础设施"升级为"核心生产资料"，行业从建设驱动转向消费驱动，商业模式的确定性显著增强。

### 6.2 标的优先级排序

| 优先级 | 标的 | 核心理由 | 适合投资者 |
|--------|------|---------|-----------|
| ⭐⭐⭐⭐⭐ | **润泽科技** | AIDC龙头，业绩确定性最强 | 稳健型 |

### 6.3 核心结论

> **算力租赁赛道**：2026年是业绩验证的关键之年。协创数据Q1业绩预告已打响第一枪，行业进入"量价齐升"的黄金窗口。但需警惕高估值下的波动风险，建议关注业绩兑现能力最强的头部标的。
>
> **AIDC赛道**：从概念叙事转向业绩验证阶段。润泽科技作为龙头已率先完成业绩兑现，AIDC收入占比快速提升。行业壁垒（能耗指标、客户资源、技术能力）正在加深，龙头优势将持续扩大。
>
> **整体策略**：**"龙头为锚，弹性为翼"**——以润泽科技、协创数据为核心持仓，以科华数据为安全边际配置，以英维克为主题弹性配置。
